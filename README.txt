Ide Aplikasi: Pemetaan Museum Di Bandung
Tujuan dibuatnya aplikasi: Tujuannya di buatnya pemetaan museum di bandung ini digunakan untuk menemukan lokasi museum yang di bandung, menyediakan informasi terkait dengan museum seperti lokasi spesifik tentang musem yang bandung 
Input data:
-	Nama Musem di Bandung
-	Alamat Museum
-	Latitude	
-	Longitude
-	Jam Buka museum
-	Jam Tutup museum
-	Biaya masuk museum	

Informasi yang ditampilkan dalah peta sebaran dalam bentuk marker, dengan informasi nama sekolah dalam pop up disetiap proses hover dititik marker. 

Tujuan dari informasi yang ditampilkan:
1.	Memberikan informasi yang jelas dan akurat tentang lokasi museum, sehingga pengguna dapat dengan mudah menemukan dan mencapai destinasi tersebut.
2.	Memberikan informasi tentang jam buka museum, jam tutup museum dan biaya masuk, sehingga pengguna dapat merencanakan kunjungan mereka dengan baik.

Pada Aplikasi pemetaan museum terdapat system CRUD yang dimana admin dapat menambahkan data museum dengan mengisi form input seperti nama museum, alamat museum, latitude, longitude, jam buka, jam tutup dan biaya masuk. Kemudian admin dapat melakukan update data musem dengan mengklik button update kemudian admin dapat mengubah data seperti nama museum, alamat museum, latitude, longitude, jam buka, jam tutup dan biaya masuk. Selanjutnya admin dapat menghapus data museum dengan mengklik button hapus. Kemudian admin dapat melihat data museum pada halaman data museum yang terdapat marker dan informasi tentang museum yang sudah di input tadi. Sebagai user/pengguna mereka dapat melihat informasi langsung berupa marker dan terdapat informasi seperti nama museum, alamat museum, latitude, longitude, jam buka, jam tutup dan biaya masuk tanpa login terlebih dahulu.
